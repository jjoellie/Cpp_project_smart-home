# Smart Home System – C++ & Qt

## Projectbeschrijving
Dit project is een Smart Home simulatie ontwikkeld in C++ met Qt Widgets.
Het systeem simuleert kamers met slimme apparaten zoals lampen, thermostaten
en bewegingssensoren die bediend kunnen worden via een GUI.

## Functionaliteiten
- Overzicht van kamers en apparaten
- Lampen aan/uit en helderheid instellen
- Thermostaat target temperatuur instellen
- Bewegingssensor simuleren en resetten
- Automatische updates via timer

## Build & Run
### Vereisten
- Qt 5.15+ of Qt 6
- Qt Creator
- C++17 compiler

### Build-instructies
1. Open Qt Creator
2. Open het project (.pro of CMakeLists.txt)
3. Selecteer een Desktop Qt Kit
4. Build het project
5. Run de applicatie
